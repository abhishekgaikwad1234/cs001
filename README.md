Grameen Light is an Android-based application designed to support rural electrification by promoting awareness, accessibility, and management of solar energy solutions. The app aims to empower rural communities by providing information about sustainable lighting options, government schemes, and easy monitoring of energy usage.
 Features
 Solar Energy Information
Learn about solar panels, installation, and benefits.
 Rural Electrification Support
Provides solutions for villages with limited or no electricity.
 Energy Usage Monitoring
Track power consumption and efficiency.
 Nearby Service Providers
Locate solar installation services and support centers.
 Government Schemes & Updates
Stay updated on subsidies and rural electrification programs.
 User-Friendly Interface
Simple UI designed for easy use by rural users.
 Tech Stack
Frontend: XML (Android UI)
Backend: Java / Kotlin
Database: Firebase / SQLite
Tools: Android Studio
